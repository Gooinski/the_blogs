Setting up venv

Author: Chris Lipinski, Published: 20221218

Python Enviroment Setup

From Python 3.4 venv can be used from python command.

To setup a venv in your working directory:

python -m venv venv

To activate venv in Windows:

venv\Scripts\activate.bat

To activate venv in Linux:

$ source myvenv/bin/activate
